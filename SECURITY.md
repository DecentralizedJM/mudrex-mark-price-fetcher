# Security Policy

## Reporting a Vulnerability

If you discover a security issue in PriceFetcher, please report it privately:

- **Email:** security@decnetralizedjm.dev
- **GitHub:** [Open a private security advisory](https://github.com/DecentralizedJM/mudrex-mark-price-fetcher/security/advisories/new) on this repository

Please include steps to reproduce, affected routes or components, and any suggested fix if you have one.

We aim to acknowledge reports within **3 business days** and will coordinate disclosure once a fix is available.

## Scope

This policy covers the PriceFetcher application in `marktrace/`, including the Express server, public API routes, and the separate `/admin` usage dashboard.

## Out of Scope

- Issues in third-party services (Mudrex API availability, Railway platform)
- Social engineering or physical attacks
- Denial-of-service attacks against production infrastructure

## Safe Harbor

Good-faith security research that follows this policy will not be pursued legally. Do not access data belonging to others, exfiltrate production telemetry, or disrupt the service.
